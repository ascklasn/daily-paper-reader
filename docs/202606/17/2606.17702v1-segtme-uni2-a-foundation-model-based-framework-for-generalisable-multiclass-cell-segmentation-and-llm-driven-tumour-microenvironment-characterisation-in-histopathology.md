## 摘要
从常规H&E染色组织学图像中表征肿瘤微环境（TME）需要同时进行细胞分割、特征提取和可解释的临床报告。我们提出SEGTME-UNI2，一个统一的框架来满足这些需求。其核心是UNI2-UPERHOVER，一种双头分割模型，将UNI2-H病理基础模型（ViT-Giant，在来自10万张切片的超过1亿张图块上预训练）与两个并行的UperNet解码器配对：一个用于六类语义分割，另一个用于水平-垂直梯度回归，从而实现基于分水岭的细胞核实例分离。为了解决大型真实世界标注库中缺乏像素级标注的问题，UNI2-UPERHOVER经历了三阶段渐进式伪标签课程。每个阶段都从头训练一个新模型而不进行权重迁移，仅通过提高伪标签质量来推动改进。第一阶段：使用人工标注的PanNuke（7901张图像，189744个细胞核，0.25微米/像素）。第二阶段：使用第一阶段模型在271711张TCGA-UT尺度0图块（0.5微米/像素）上经过熵筛选的伪标签。第三阶段：使用第二阶段模型在所有1608060张TCGA-UT图块上（六个分辨率尺度，0.5-1.0微米/像素）的伪标签。分割输出输入到结构化TME特征提取流水线，计算20多个每图块的组成、形态、空间熵和细胞间距离指标。这些指标编码为JSON并传递给微调后的NVIDIA BioNeMo GPT模型，以生成临床可解释的TME叙述。在保留的PanNuke和TCGA-UT分区上的初步验证证明了框架的可行性和内部一致性。伪标注的TCGA-UT数据集和UNI2-UPERHOVER检查点已公开发布，以支持大规模TME分析和空间生物学研究。

## Abstract
Characterising the tumour microenvironment (TME) from routine H&E-stained histology images requires simultaneous cell segmentation, feature extraction, and interpretable clinical reporting. We present SEGTME-UNI2, a unified framework addressing these requirements. Its core is UNI2-UPERHOVER, a dual-head segmentation model pairing the UNI2-H pathology foundation model (ViT-Giant, pretrained on >100M tiles from 100K slides) with two parallel UperNet decoders: one for six-class semantic segmentation and one for horizontal-vertical gradient regression enabling watershed-based nuclear instance separation. To address the lack of pixel-level annotations in large real-world repositories, UNI2-UPERHOVER undergoes a three-stage progressive pseudo-label curriculum. Each stage trains a fresh model without weight transfer, driving improvement entirely via increased pseudo-label quality: Stage 1: Uses human-annotated PanNuke (7,901 images, 189,744 nuclei, 0.25 um/pixel). Stage 2: Uses entropy-filtered pseudo-labels from the Stage 1 model on 271,711 TCGA-UT scale-0 patches (0.5 um/pixel). Stage 3: Uses pseudo-labels from the Stage 2 model on all 1,608,060 TCGA-UT patches across six resolution scales (0.5-1.0 um/pixel). Segmentation outputs feed a structured TME feature extraction pipeline computing 20+ per-patch compositional, morphological, spatial entropy, and intercellular distance metrics. These are encoded as JSON and passed to a fine-tuned NVIDIA BioNeMo GPT model to generate clinically interpretable TME narratives. Preliminary validation on held-out PanNuke and TCGA-UT partitions demonstrates framework feasibility and internal consistency. The pseudo-labelled TCGA-UT dataset and UNI2-UPERHOVER checkpoint are publicly released to support large-scale TME profiling and spatial biology research.

- **数据鸿沟**：TCGA图像仓库含超过30,000患者、33种癌症类型的全切片图像，但完全没有像素级标注。如何利用海量未标注数据弥补标注稀缺性是核心挑战。
- **整体含义**：作者提出一个端到端的统一框架，利用大规模病理基础模型（UNI2-H）、渐进式伪标签课程、结构化TME特征提取和大语言模型（BioNeMo GPT），旨在实现**可泛化的多类细胞分割**与**临床可读的TME叙述生成**，从而推动空间生物学研究和临床决策支持。

## 2. 论文提出的方法论

### 核心思想
- 以 **UNI2-H**（ViT-Giant，在超过1亿图块、10万张切片上预训练）作为共享编码器，搭配**两个独立的UperNet解码器**（语义头与HV回归头），实现六类语义分割与基于分水岭的核实例分离。
- 通过**三阶段渐进式伪标签课程**，逐步从人工标注的PanNuke扩展到160万张TCGA-UT多尺度图块，无需额外人工标注。
- 分割输出经过结构化特征提取（20+指标），再由微调的**BioNeMo GPT**模型生成自然语言TME描述。

### 关键技术细节
1. **UNI2-UperHoVer 架构**：
   - 编码器：UNI2-H（ViT-Giant），从第5、11、17、23个Transformer块提取四层特征金字塔（对应1/4、1/8、1/16、1/32空间尺度），通过1×1卷积投影至FPN标准通道数（256/512/1024/2048）。
   - 两个UperNet解码头（参数不共享）：
     - **语义头**：输出6通道类概率（PanNuke类别：Neoplastic、Inflammatory、Connective、Dead、Non-neoplastic Epithelial、Background），使用交叉熵损失（忽略索引255）。
     - **HV回归头**：输出2通道水平/垂直梯度图，损失函数为 **MSE + 2×MSGE**（MSGE为梯度匹配项，惩罚边界处斜率误差），最终综合损失：\( \mathcal{L}_{\text{total}} = \mathcal{L}_{\text{sem}} + \lambda \mathcal{L}_{\text{hv}}, \lambda=1.0 \)。
   - **动态HV目标合成**：从语义掩码（仅像素级类别标签）通过连通组件分析生成HV梯度伪标签，无需实例级地面真值。方法：二值化→连通组件标签→计算每组件质心→对组件内每个像素计算归一化相对位置：\( T_x = \text{clip}((x - \hat{c}_x)/10, -1, 1) \)，\( T_y = \text{clip}((y - \hat{c}_y)/10, -1, 1) \)。这使训练可以仅依赖语义伪标签。

2. **三阶段渐进伪标签课程**：
   - **Stage 1**（有监督种子）：在PanNuke（7,901图块，189,744核，0.25 μm/pixel）上端到端训练UNI2-UperHoVer。得到模型 \( \Phi_1 \)。
   - **Stage 2**（尺度0适应）：使用 \( \Phi_1 \) 对TCGA-UT scale-0（0.5 μm/pixel，271,711图块）推理，计算每像素熵，只保留每个图块平均熵低于第70百分位阈值的图块（top-70%最自信），得到伪标签。然后从UNI2-H重新初始化，训练新模型 \( \Phi_2 \)。
   - **Stage 3**（全尺度扩展）：使用 \( \Phi_2 \) 对所有6个分辨率尺度（0.5–1.0 μm/pixel，1,608,060图块）生成熵过滤伪标签，再次从UNI2-H重初始化训练最终模型 \( \Phi_3 \)。
   - 关键：每阶段不使用前阶段权重，改进完全来自伪标签质量提升。

3. **TME特征提取流水线**：
   - 输入：实例掩码（region IDs）和语义掩码（0–5类）。
   - 计算：每类细胞计数、人口比例、平均核面积；肿瘤-间质分数 \( f_{TS} = N_{\text{neo}} / (N_{\text{neo}} + N_{\text{conn}}) \)，免疫浸润分数 \( f_{II} = N_{\text{inf}} / (N_{\text{neo}} + N_{\text{inf}}) \)；使用cKDTree计算免疫细胞→肿瘤细胞最小/平均距离、邻近50px内的免疫细胞数量；基于6-最近邻的细胞类型香农熵 \( H_i = -\sum_c p_{i,c} \log_2 p_{i,c} \)，然后取全局平均。
   - 输出：结构化JSON（含原始数据+六个表型轴解释标签+扁平化生物标记词列表）。

4. **BioNeMo GPT 监督微调**：
   - 将TME特征JSON序列化为固定模板提示语（TME features: ... Phenotype labels: ... Biological tokens: ... Narrative:），微调BioNeMo GPT以自回归生成Narrative字段。
   - 训练目标：标准交叉熵损失仅针对叙述部分。
   - 作用：生成临床可读的TME描述，且每个陈述可追溯至定量特征值，降低幻觉风险。

## 6. 论文的主要结论与发现

- **框架可行**：提出并初步验证了SegTME-UNI2框架能够从H&E图像端到端实现多类细胞分割、TME特征提取、临床叙述生成。
- **渐进伪标签课程有效**：三阶段训练显示mIoU逐步提升（初始mIoU从0.41→0.47→0.53），证明随伪标签质量提升，模型无需继承前阶段权重也能改进。
- **UperNet适宜多尺度场景**：UperNet的PPM+FPN融合在处理0.25–1.0 μm/pixel的尺度变化时优于点式解码器。
- **BioNeMo降低幻觉**：基于结构化特征而非图像像素生成叙述，提供了可追溯的临床解释。
- **公开资源**：发布了最大规模核级伪标注TCGA数据集（160万图块）和三个模型权重，促进可重复研究。

## 7. 优点

- **创新性整合**：首次将病理基础模型（UNI2-H）、双头分割、渐进伪标签课程、结构化特征提取与LLM叙事生成统一在一个框架中。
- **解决标注瓶颈**：三阶段课程从有限人工标注扩展到超大规模未标注数据，无需额外人工成本，方法可推广到其他医学图像领域。
- **动态HV目标合成**：创新性地从纯语义伪标签合成实例分离所需的HV梯度，绕过了实例标注的稀缺性，使伪标签监督成为可能。
- **UperNet选择合理**：基于多尺度金字塔池化，天然适合不同分辨率间训练-推理偏移，论文给出了明确理由。
- **可解释性与鲁棒性**：叙述基于聚合统计量而非单个细胞边界精度，在染色变异大时仍能保持临床相关性（如Case D浅粉红病例）。
- **开放性**：代码、模型、数据集均公开发布（HuggingFace、PyPI），支持社区验证和扩展。

## 8. 不足与局限

- **缺乏外部定量验证**：未在MoNuSeg、CoNSeP等标准基准上与其他方法对比，难以评估实际性能优劣。这是最显著的不足。
- **消融实验缺失**：无法量化HV头、FPN深度、熵阈值、课程阶段数等组件的单独贡献，影响结论的说服力。
- **伪标签质量未验证**：TCGA-UT上的伪标签未与任何人工标注进行直接比较（虽然有定性观察），可能包含系统性偏差或错误，影响下游适用性。
- **统计严谨性不足**：结果仅来自单次训练，未报告置信区间或多重随机种子结果，难以判断泛化稳定性。
