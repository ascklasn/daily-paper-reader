## 摘要
从常规H&E染色的组织学图像中表征肿瘤微环境（TME）需要同时进行细胞分割、特征提取和可解释的临床报告。我们提出了SEGTME-UNI2，一个统一框架来满足这些需求。其核心是UNI2-UPERHOVER，一个双头分割模型，将UNI2-H病理学基础模型（ViT-Giant，在来自10万张切片的超过1亿个图块上预训练）与两个并行的UperNet解码器配对：一个用于六类语义分割，另一个用于水平-垂直梯度回归，从而实现基于分水岭的细胞核实例分离。为了解决大型真实世界存储库中缺乏像素级标注的问题，UNI2-UPERHOVER经历了三阶段渐进式伪标签课程。每个阶段都训练一个全新的模型而不进行权重迁移，完全通过提高伪标签质量来驱动改进：阶段1：使用人工标注的PanNuke（7,901张图像，189,744个细胞核，0.25微米/像素）。阶段2：在271,711个TCGA-UT尺度0图块（0.5微米/像素）上，使用阶段1模型的熵过滤伪标签。阶段3：在全部1,608,060个TCGA-UT图块（跨越0.5-1.0微米/像素的六个分辨率尺度）上，使用阶段2模型的伪标签。分割输出送入一个结构化的TME特征提取流程，计算20多个每图块成分、形态、空间熵和细胞间距离指标。这些指标编码为JSON并传递给微调的NVIDIA BioNeMo GPT模型，以生成临床可解释的TME叙述。在保留的PanNuke和TCGA-UT分区上的初步验证表明框架可行且内部一致。伪标记的TCGA-UT数据集和UNI2-UPERHOVER检查点已公开发布，以支持大规模TME分析和空间生物学研究。

## Abstract
Characterising the tumour microenvironment (TME) from routine H&E-stained histology images requires simultaneous cell segmentation, feature extraction, and interpretable clinical reporting. We present SEGTME-UNI2, a unified framework addressing these requirements. Its core is UNI2-UPERHOVER, a dual-head segmentation model pairing the UNI2-H pathology foundation model (ViT-Giant, pretrained on >100M tiles from 100K slides) with two parallel UperNet decoders: one for six-class semantic segmentation and one for horizontal-vertical gradient regression enabling watershed-based nuclear instance separation. To address the lack of pixel-level annotations in large real-world repositories, UNI2-UPERHOVER undergoes a three-stage progressive pseudo-label curriculum. Each stage trains a fresh model without weight transfer, driving improvement entirely via increased pseudo-label quality: Stage 1: Uses human-annotated PanNuke (7,901 images, 189,744 nuclei, 0.25 um/pixel). Stage 2: Uses entropy-filtered pseudo-labels from the Stage 1 model on 271,711 TCGA-UT scale-0 patches (0.5 um/pixel). Stage 3: Uses pseudo-labels from the Stage 2 model on all 1,608,060 TCGA-UT patches across six resolution scales (0.5-1.0 um/pixel). Segmentation outputs feed a structured TME feature extraction pipeline computing 20+ per-patch compositional, morphological, spatial entropy, and intercellular distance metrics. These are encoded as JSON and passed to a fine-tuned NVIDIA BioNeMo GPT model to generate clinically interpretable TME narratives. Preliminary validation on held-out PanNuke and TCGA-UT partitions demonstrates framework feasibility and internal consistency. The pseudo-labelled TCGA-UT dataset and UNI2-UPERHOVER checkpoint are publicly released to support large-scale TME profiling and spatial biology research.

### 1. 论文的核心问题与整体含义（研究动机和背景）

- **核心问题**：从常规H&E染色的组织学图像中表征肿瘤微环境（TME），需要同时完成**多类细胞分割**（语义分割+实例分离）、**定量特征提取**（如细胞计数、空间熵、免疫-肿瘤距离）和**可解释临床报告**生成。现有方法存在以下瓶颈：
  - 强监督实例分割模型（如HoVer-Net）依赖昂贵的像素级多边形标注，仅适用于小规模数据集（如PanNuke，7,901张图像），缺乏泛化能力，无法直接应用于大规模真实世界存储库（如TCGA，160万张图块）。
  - 语义分割模型（如UperNet）能提供组织级上下文，但无法分离彼此接触的细胞核。
  - 两类模型均不直接产生临床可读的输出，医生难以直接从定量数值中快速理解TME状态。
- **整体含义**：本文旨在弥合这三个需求之间的鸿沟，提出一个统一的端到端框架 **SegTME-UNI2**，它结合大规模病理基础模型（UNI2-H）、一种无需实例级标注即可进行多类细胞分割的双头架构、一种可扩展到1.6M无标注图块的渐进式伪标签课程，以及一个基于LLM（BioNeMo GPT）的临床叙述生成管道，从而实现从H&E图像到临床解释性报告的自动化流程。

### 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：利用**预训练的病理基础模型（UNI2-H, ViT-Giant）**作为共享编码器，在其上附加**两个独立的UperNet解码器头**——一个用于六类语义分割（背景、肿瘤、炎症、结缔、坏死、上皮），另一个用于水平-垂直（HV）梯度回归；通过**动态HV目标合成**技术从纯语义掩码中自动合成HV梯度目标，从而摆脱对实例级标注的依赖；进一步通过**三阶段渐进式伪标签课程**，从少量人工标注（PanNuke）逐步扩张到大规模无标注TCGA数据（160万图块，6种分辨率尺度），每阶段均从UNI2-H预训练权重**全新初始化训练**（不继承上一阶段权重），仅依靠前一阶段生成的熵过滤伪标签提升质量。
- **关键技术细节**：
  - **UNI2-UPERHOVER双头分割模型**：UNI2-H（ViT-Giant, 24层，隐藏维度1536）提取四层特征金字塔（来自Transformer块5/11/17/23，对应1/4, 1/8, 1/16, 1/32空间步长），通过1×1卷积投影到标准FPN通道数（256/512/1024/2048）。每个UperNet解码器包含金字塔池化模块和FPN融合，语义头输出6通道logits，HV回归头输出2通道HV图（无激活函数）。
  - **动态HV目标合成**：每个训练迭代中，对当前语义掩码（仅需语义标签）执行连通组件分析，计算每个组件质心，然后对每个像素生成归一化到[-1,1]的HV目标（质心距离除以10并clip）。该过程在`torch.no_grad()`内完成，几乎无额外开销。这使得后续伪标签阶段也能同时生成语义和HV目标，无需任何实例级人类标注。
  - **损失函数**：语义交叉熵（忽略索引255）；HV回归损失 = 像素级MSE + 2 × 梯度匹配MSE（Sobel算子计算空间梯度）。总损失 = ℒ_sem + 1.0 × ℒ_hv。
  - **三阶段伪标签课程**：
    - Stage 1：在PanNuke（0.25 μm/px）上训练250轮，得到初始模型。
    - Stage 2：Stage 1模型在TCGA-UT尺度0（0.5 μm/px, 271,711个图块）上推理，计算每个像素熵，仅保留平均熵低于70百分位数的低不确定性图块作为训练集。用这些伪标签从UNI2-H重新初始化训练250轮。
    - Stage 3：Stage 2模型在TCGA-UT所有6个尺度（0.5-1.0 μm/px, 1,608,060个图块）上生成熵过滤伪标签，同样从UNI2-H重新初始化训练100轮（每轮图块数大，总梯度步数更多）。最终模型为Stage 3。
  - **TME特征提取管道**：从分割输出（实例掩码+语义图）计算每图块的组成（每类细胞计数/比例/平均面积）、临床比率（肿瘤-间质比、免疫浸润比）、空间交互（基于cKDTree的免疫-肿瘤最小/平均距离、50px内接触免疫细胞数）、空间熵（每个细胞周围5个邻居的香农熵均值）。输出为结构化JSON。
  - **生物令牌翻译与BioNeMo叙述生成**：JSON中的数值经规则阈值（表4）映射为6个表型轴（组成、免疫状态、间质状态、空间交互、架构、形态）的类别标签和一组扁平生物学令牌（如“Immune Hot”“Immune exclusion”）。令牌列表输入微调的NVIDIA BioNeMo GPT模型（基于SFT，损失仅计算叙述部分），生成临床可读的TME描述（例如：“The section exhibits low tumour purity ... immune cells spatially excluded from the tumour core ...”）。

### 6. 论文的主要结论与发现

- **主要结论**：SegTME-UNI2框架可行，其核心创新（双头分割+动态HV合成+三阶段伪标签课程）能够在不增加人工标注的前提下，将分割监督从7,901张PanNuke图像扩展到160万张TCGA-UT图块。三个阶段的渐进式训练使每阶段的起点mIoU逐步提升，证明了伪标签质量的改善。最终模型（Stage 3）在TCGA-UT多尺度数据上实现了合理的mIoU（0.772，基于自身伪标签验证）。通过BBioNeMo集成，定量TME特征可转化为临床可读的叙述，且由于基于聚集统计而非单细胞精度，对分割不完美具有一定的鲁棒性。作者公开发布了最大规模的细胞级伪标注TCGA-UT数据集和模型权重，以推动大规模TME分析。

### 7. 优点：方法或实验设计上的亮点

1. **创新性双头架构**：首次将UperNet解码器与病理基础模型（UNI2-H）结合用于多类细胞分割和实例分离，且通过动态HV目标合成摆脱了对实例标注的依赖，使伪标签课程可行。
2. **渐进式伪标签课程设计**：阶段之间不共享权重，仅依靠前阶段伪标签质量提升，避免了错误累积；先同分辨率（PanNuke 0.25→TCGA-UT 0.5），再扩展到更粗尺度，符合学习排序原则。
3. **模块化且鲁棒的TME特征-叙述管道**：从分割→统计特征→规则性生物令牌→LLM生成，每一层都具备可解释性。LLM的提示基于定量值，而非直接图像像素，显著降低幻觉风险；同时，基于统计聚合特征而非单细胞边界精度，使得即使分割质量较差时（如浅色染色），整个管道的临床描述仍能保持合理。
4. **完全开源**：提供了三个模型检查点、重新组织的TCGA-UT伪标签数据集（HuggingFace）和pip包，增强了可复现性和后续研究的门槛。
5. **跨机构定性验证**：选择4个非TCGA域的HNSCC案例，涵盖不同染色和TME表型，展示了模型的泛化性和鲁棒性，尽管只是定性。

### 8. 不足与局限：包括实验覆盖、偏差风险、应用限制

- **严重缺乏定量比较与消融**：未与当前SOTA方法（HoVer-Net、CellPose、StarDist）在公共基准上公平对比，无法评估相对性能；缺少任何消融实验（如HV回归头贡献、FPN层数、熵阈值、阶段数量等），所以不知道每个组件的实际增益。
- **伪标签质量未验证**：TCGA-UT的伪标签没有人类专家验证（仅计划200张），模型自身测试集也是伪标签，存在标签噪声自我放大的风险；mIoU指标可能高估真实精度。
- **统计显著性缺失**：仅呈现单次运行曲线，没有多次随机种子取平均和置信区间，结果可靠性存疑。
- **动态HV目标合成的理论缺陷**：对于同一类且互相邻接的细胞核（如大量肿瘤细胞聚集成簇），连通组件分析会将其合并为一个假核，导致HV目标错误；作者只提及“abutting same-class nuclei”问题，并计划未来引入形态学预分离或学习细化模块。
- **Tile边界伪影**：使用“last-write”策略拼接HV图，会在图块边界产生梯度不连续，影响分水岭分割；计划采用高斯羽化混合但未实现。
- **生物令牌的规则依赖**：阈值（如免疫热>0.30、空间熵>0.70）来源于文献，缺乏数据驱动优化；不同染色/病理可能产生偏移，且未讨论阈值敏感性。
